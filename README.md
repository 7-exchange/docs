7.Exchange connects DEXs, bridges, wallets, and liquidity sources into a single non-custodial flow, with routes optimized for real execution outcomes rather than headline quotes. This repo holds the source for those docs.

## What's inside

- **Getting Started** — what 7.Exchange is, connecting a wallet, your first swap
- **Core Concepts** — routing, cross-chain execution, quotes & pricing, slippage & MEV protection, fees
- **Features** — cross-chain swaps, direct swap, limit orders, perpetual trading, transaction tracking, referrals
- **Supported Infrastructure** — chains, tokens, bridges & liquidity sources, wallets
- **Security** — [security model](security/security-model.md) and [bug bounty program](security/bug-bounty-program.md)
- **Developers** — [overview](developers/overview.md), [API reference](developers/api-reference.md), [SDK](developers/sdk.md), [widget](developers/widget.md), [partner integration](developers/partner-integration.md)
- **Resources** — roadmap, FAQ, glossary, brand assets, community links
